# 🥞 Pancake Shop App
Ứng dụng đặt bánh Pancake xinh xắn được xây dựng bằng **Flutter & Firebase**.

## ✨ Tính năng nổi bật
* **Trang chủ:** Hiển thị danh sách bánh Pancake đa dạng.
* **Yêu thích ❤️:** Thêm/Xóa món bánh vào danh sách yêu thích (Sử dụng Provider).
* **Giỏ hàng 🛒:** Quản lý món ăn, tính tổng tiền chuyên nghiệp.
* **Chi tiết sản phẩm:** Xem thông tin chi tiết từng loại bánh.

## 🛠️ Công nghệ sử dụng
* Flutter SDK & Dart
* Provider (State Management)
* Firebase Authentication & Firestore
